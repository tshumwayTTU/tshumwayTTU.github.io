# Project Scope Statement

## Project Purpose
The purpose of this project is to create a single-page personal website using semantic HTML to apply the scoping principles from Chapter 3. By defining the scope before writing code, I can stay focused and avoid unnecessary work. It will ensure I am creating a well-defined and well-structured plan.

## Description
I will build a well-structured `index.html` page that introduces who I am, my professional interests, course goals, and hobbies. The page will use semantic HTML elements including headings, paragraphs, lists, and hyperlinks to organize content clearly. No external styling or frameworks will be used as the focus is on proper document structure.

## Desired Results
- A valid HTML document with proper structure.
- A main heading with my name and a subheading for my professional interests.
- At least two paragraphs describing my goals for the course or career.
- An unordered list with at least three hobbies or interests.
- A working hyperlink to my GitHub profile.

## Exclusions
- No CSS styling or visual design beyond default browser rendering.
- No JavaScript or interactive functionality.
- No images or multimedia content.
- No multi-page navigation or external libraries.
