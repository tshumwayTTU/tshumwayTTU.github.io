# Timothy Shumway — Personal Website

A single-page personal website introducing me as a software developer, built with semantic HTML and custom CSS.

## Live Site

Deployed at: [https://tshumwayttu.github.io](https://tshumwayttu.github.io)

## About This Project

This website was built to apply scoping and planning principles learned in coursework while practicing the web development cycle. It introduces my background, professional goals, and personal interests in a clear and professional format. It serves as a learning exercise and lightweight introduction for anyone who wants to know more about who I am and what I'm working on.

## Built With

- **HTML5** — semantic markup for structure
- **CSS3** — custom styling
- **GitHub Pages** — hosting service

## Features

- Semantic HTML structure
- Responsive full-height layout
- Custom typography 

## Project Structure


## Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/tshumwayTTU/tshumwayTTU.github.io.git
   ```
2. Navigate into the project folder:
   ```
   cd tshumwayTTU.github.io
   ```
3. Open `index.html` in your browser.
